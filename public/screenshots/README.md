# App screenshots

Copy your App Store screenshot images into this folder.

## Filenames in use

- `01.png` through `07.png`

PNG or JPG both work. Use the same aspect ratio as your App Store assets (iPhone screenshots are typically portrait).

## After adding or changing files

1. Drop image files into this folder (`public/screenshots/`).
2. Update the `appScreenshots` list in `src/lib/constants.ts` if filenames change.
3. Refresh the site — they appear in the **See Rivit in action** section on the homepage.

You do **not** paste images into the Cursor chat. Add them as files in this folder instead.
